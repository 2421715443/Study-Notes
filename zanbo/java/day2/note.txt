程序的执行结构：
1.顺序
2.分支
3.循环

循环的分类:
一、while循环：相对于for循环要灵活，因为3.改变控制循环变量的值可以方法体的任意位置
1.控制循环的变量
while(2.条件表达式){
    2为true执行的代码
	3.改变控制循环变量的值
}

二、do...while循环  和 while的唯一区别就是至少执行一次
1.控制循环的变量
do{
    2为true执行的代码
	3.改变控制循环变量的值

}while(2.条件表达式);

三、for循环  1.最适合开发有限次数的循环 2.可读性好
for(1.控制循环的变量;2.条件表达式;3.改变控制循环变量的值){
   2为true执行的代码
}
四、关键字break和continue
break:退出当前循环；跳出当前分支
continue:退出当前循环的“本次”循环
